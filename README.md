## 【学習用】Phaser 4 サンプルゲーム

このリポジトリは、Phaser の学習を目的として作成したサンプルゲームです。

実装の出発点として、以下の記事で紹介されているゲームのソースコードを参考にしました。

- 参考元: https://qiita.com/y_o_28/items/6a1c1cd01cfad1efe37b

参考元のコードは Phaser 3 を前提とした構成だったため、本リポジトリでは Phaser 4 で動作するように書き直しています。

主な変更点は以下の通りです。

- Phaser 4 の型定義に合わせて、`Group`、`TextStyle`、`collider` / `overlap` まわりの書き方を修正
- `Scene` に集中していた処理を、プレイヤー、地形、スター、ボム、スコア表示などの役割ごとに分離
- アセットキー、座標、スコア、物理設定などの固定値を定数として整理
- 初学者が処理の流れを追いやすいように、各ファイルへコメントを追加

元のゲームの挙動はできるだけ保ちつつ、Phaser 4 と TypeScript で読みやすく拡張しやすい構成になるようにリファクタリングしています。
